## Installation

### 1. Clone this repo

Via HTTPS:

```bash
git clone https://github.com/Xavier2p/dotfiles.git
```

Via SSH:
```bash
git clone git@github.com:Xavier2p/dotfiles.git
```

### 2. Run the install script

```bash
cd dotfiles
make DOTFILES=/your/install/path
```

### 3. Reload your shell

```bash
source ~/.zshrc
```
