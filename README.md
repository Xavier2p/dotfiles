## Installation

### 1. Clone this repo

#### Via HTTPS

```bash
git clone https://github.com/Xavier2p/dotfiles.git ~/.config/dotfiles
```

#### Via SSH

```bash
git clone git@github.com:Xavier2p/dotfiles.git ~/.config/dotfiles
```

### 2. Run the install script

```bash
cd ~/.config/dotfiles
make
```

### 3. Reload your shell

```bash
source ~/.zshrc
```

### 5. Enjoy!
